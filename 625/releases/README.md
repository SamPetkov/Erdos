# Release archives

- `Erdos-625-complete-dossier-2026-07-14.zip` is the current complete package.
  It contains the intended `625/` dossier: the canonical and publication-layout
  manuscripts, TeX/BibTeX/PDF artifacts, the complete `arxiv/` source library,
  verification and audit records, reproducibility scripts and assets, and the
  pinned Lean project with all formalization source files and milestone notes.
- `Erdos-625-complete-dossier-2026-07-13.zip` is the previous complete package
  and predates the arXiv source library and packaged Lean source checkpoint.
- `Erdos-625-complete-dossier-2026-07-12.zip` is retained as a historical
  pre-adversarial-audit snapshot.
- `Erdos-625-proof-dossier-2026-07-12.zip` is the earlier pre-verification
  package and is retained only as a historical snapshot.

Nested ZIPs, `tmp/`, Lean build/service caches (`.lake/` and `.aristotle/`),
Python caches, and copyrighted historical-source PDFs are excluded.  The
copyrighted originals remain local and are represented only by bibliographic
and audit metadata.  `SHA256SUMS.txt` is intentionally outside the complete
archive so it can hash the archive itself.
