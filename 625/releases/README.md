# Release archives

- `Erdos-625-complete-dossier-2026-07-13.zip` is the current complete package.
  It contains the full `625/` dossier, including verification assets, TeX,
  PDF, static preview, README animation (GIF and MP4), animation provenance,
  audits, sources, and reproducibility files. Nested ZIP binaries are excluded
  to avoid recursive packaging.
- `Erdos-625-complete-dossier-2026-07-12.zip` is the previous complete package
  and is retained as a historical pre-adversarial-audit snapshot.
- `Erdos-625-proof-dossier-2026-07-12.zip` is the earlier pre-verification
  package and is retained only as a historical snapshot.

Repository copies are accompanied by `SHA256SUMS.txt`.  That checksum file is
intentionally outside the complete archive so it can hash the archive itself.
