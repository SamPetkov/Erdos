# arXiv preprints

This directory contains submission-oriented copies of the manuscripts for
Erdős Problems 593 and 625.

- `arxiv_593/`: TeX source, bibliography, generated bibliography, and PDF.
- `arxiv_625/`: TeX source, bibliography, generated bibliography, and PDF.
- `arxiv_593.pdf` and `arxiv_625.pdf`: convenient top-level copies.

The PDFs carry fixed manuscript dates: 11 July 2026 for Problem 593 and
12 July 2026 for Problem 625. arXiv supplies separate public submission and
revision dates in its own record.
